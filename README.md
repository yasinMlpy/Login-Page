# Login Form

A simple login form built with HTML and CSS.

## Features
- Responsive login box centered on the page
- Background image
- Rounded corners and glow effect using `box-shadow`
- Styled labels and input fields
- Hover effect on the login button
- Device selection using a dropdown menu

## Technologies Used
- HTML5
- CSS3
